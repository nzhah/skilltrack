A full-stack web app for developers to track skills, monitor progress, and visualize learning growth.

## Features
- **Authentication:** JWT-based secure login/register
- **Skill Management:** Add, edit, delete skills
- **Progress Tracking:** Monitor proficiency and progress
- **Dashboard:** Visual stats and charts
- **Responsive UI:** Works on desktop & mobile

## Tech Stack
- **Frontend:** React, React Router, Axios, Recharts, CSS3
- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT
- **Deployment:** MongoDB Atlas

## Quick Start
```bash
# Clone repo
git clone <your-repo-url>
cd skilltrack

# Backend
cd server
npm install
npm run dev

# Frontend
cd ../client
npm install
npm start
